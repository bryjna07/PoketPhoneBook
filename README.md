## 📱 주요 기능

1. **랜덤 포켓몬 뽑기**  
   연락처에 등록할 포켓몬 사진을 랜덤으로 선택할 수 있습니다.

2. **연락처 기능**  
   저장할 연락처를 등록, 수정 및 삭제할 수 있습니다.
